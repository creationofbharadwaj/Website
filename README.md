# Thatha Ennai - Ayurvedic Pain Reliever Website

A conversion-focused, modern business website for Thatha Ennai Ayurvedic Pain Reliever oil.

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion

## Getting Started

### Prerequisites

- Node.js 18+ and npm

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

### Build

```bash
npm run build
npm start
```

## Features

- 10 sections: Hero, Problem, Ingredients, How It Works, Benefits, Practitioner, Testimonials, Pricing, FAQ, Footer
- Sticky WhatsApp CTA button
- Fully responsive (mobile-first)
- SEO optimized with metadata and Open Graph
- Smooth scroll animations via Framer Motion
